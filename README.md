# 🧠 Neuro Web Scan MRI

A web-based application designed to assist in MRI scan analysis using Artificial Intelligence and Machine Learning techniques. The system provides an intuitive interface for uploading MRI images and generating predictions to support medical image analysis.

## 📌 Project Overview

Neuro Web Scan MRI is an AI-powered application that helps analyze MRI scan images. The project combines a modern web interface with a machine learning backend to process medical images and provide intelligent insights.

## 🚀 Features

* MRI Image Upload
* AI-Based Scan Analysis
* User-Friendly Web Interface
* Fast Prediction Results
* Frontend and Backend Integration
* Responsive Design

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Sanjaynmp/Neuro-Web-Scan-MRI-.git
```

### Run Backend

```bash
cd backend
python app.py
```

### Run Frontend

```bash
cd frontend
npm install
npm start
```
